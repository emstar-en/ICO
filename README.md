# ICO: Intrinsic Curvature Oriented Substrate

**ICO** stands for **Intrinsic Curvature Oriented** substrate. It is a geometry-native specification spanning Euclidean, Hyperbolic, and Spherical spaces.

It is **NOT** an Initial Coin Offering.

This directory contains the core specifications for the geometry substrate used by HyperSync.

## Contents
*   `ICO_CORE_SPEC.md`: The core specification document defining the mathematical models, data types, and operations for the substrate.
